# UI-forbedringer — backlog

## Visuelt board
- Tilføj kort til rød tråd direkte fra visuelt tab (ikke kun board-tab)
- Rød tråd: drag-and-drop forbindelser på canvasset — mange-til-mange i stedet for lineær kæde
- Hover-tooltip på kort: fold indhold ud ved hover som alternativ til sidebar-klik

## Spiller
- Spiller-token: upload profilbillede som token, vises som markør på visuelt board
- Presence: live musemarkører — spillerne ser hinandens position på canvasset i realtid

## Platform / handler
- Arkivér operation: flyt hele boardet til "Afsluttede operationer" og start fresh på ny operation — inkl. logik for private noter koblet til arkiverede kort
- Landingpage: in-character kodeord der skal løses eller spores på siden for at komme videre
- Terminal-kort interaktivitet: spillerne kan selv skrive kommandoer i terminal-korttypen, handler definerer gyldige kommandoer og svar

## Teknisk gæld
- Reveal interrupt (se REVEAL_PROBLEM.md) — realtime-levering af handler-reveals til spillere er ikke løst
